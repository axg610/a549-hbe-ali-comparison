A549-HBE-ALI comparison analysis

Alignment and DEA

Raw datasets: A549, primary hBECs (submersion culture, HBE), and primary hBECs (air-liquid interface culture, ALI) treated with IL1B, glucocorticoid, or both at 6h. This is actually three experiments at different timepoints, so for now we use their shared timepoint (6 h post-treatment).

Alignment performed with kallisto version 0.46.1 and normalization/DEA performed with sleuth version something on the cluster. Everything documented in kallisto-sleuth.md. I used individual sleuth runs for each cell type, modeled as ~treatment. Effect sizes (“log2fold”) and FDR are from Wald test using unstimulated (NS) samples as baseline. I also modeled everything together under model ~treatment*condition for interaction testing. Finally, raw tpm values were extracted.

IN PROGRESS: Understanding individual variation in gene expression

Previously, we saw that the IL1B-regulated gene signature in A549 cells is maybe only ~30% shared with the primary cell types. Venn diagram for “significant” genes (log2fold <= -1 or >= 1 & FDR < 0.05) in each cell type:

One possible reason: A549s are derived from one individual, so four sequencing replicates may have low variability + higher significance. Each sequencing replicate for primary cells is a different individual, so higher variability + lower significance. Ignoring FDR entirely (i.e., only using the abs(log2fold) >= 1 threshold), A549 and primary cells actually have similarly sized regulated gene pools (panel A). In A549s, variation in tpm-derived log2fold values tend to be lower (panel B) and a greater proportion of the rep pool meets log2fold thresholds (panel C).
